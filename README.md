# Kamilyan

Kamilyan is a transformation advisory firm that helps organizations assess their current state, define a future vision, and build practical roadmaps for technology and digital transformation.

This repository contains both the Kamilyan website application and the supporting documentation that guides the project's strategy, design, and implementation.

The documentation within this repository is considered the project's canonical source of truth for strategic, design, and architectural decisions.

---

## Repository Structure

```text
Kamilyan/
├── website/        # Next.js application
├── docs/           # Project documentation
├── notes/          # Working notes and brainstorming
├── assets/         # Branding, mockups, and design assets
└── README.md

```
