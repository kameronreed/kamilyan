# Kamilyan Project Decisions

---

## 2026-07-03

### Homepage Structure

**Decision**

The homepage will include both:

- The Kamilyan Advisory Framework

- A dedicated Advisory Services section

**Reason**

The Framework explains **how Kamilyan works**.

The Services section explains **what Kamilyan offers**.

The Framework describes Kamilyan's methodology. The Services section describes Kamilyan's offerings.

Keeping both sections improves user understanding, supports SEO, and clearly communicates Kamilyan's advisory capabilities.

The Services section will retain a balanced three-card layout rather than replacing it with a text-only presentation.

**Status**

✅ Approved

### Advisory Services Categories

**Current Direction**

The homepage will present three high-level advisory service categories.

Current working candidates:

1. Digital Commerce Strategy

2. Technology Transformation

3. Advisory & Roadmapping

Each service will include:

- A concise description

- A supporting list of representative capabilities

The Services section complements the Kamilyan Advisory Framework rather than duplicating its four phases.
