# Kamilyan

Kamilyan is a transformation advisory firm focused on helping organizations assess their current state, define a future vision, and build practical roadmaps for technology and digital transformation.

This repository contains both the website application and the supporting project documentation.

---

## Repository Structure

```text
Kamilyan/
├── website/        # Next.js application
├── docs/           # Project documentation
├── notes/          # Working notes and brainstorming
└── assets/         # Branding, mockups, and design assets

```

---

## Getting Started

If you're contributing to this project or starting a new development session, read these documents in order:

1. `docs/PLAYBOOK.md` — Vision, brand, design principles, and development philosophy.
2. `docs/PROJECT_STATUS.md` — Current sprint, progress, and upcoming work.
3. `docs/DECISIONS.md` — Approved project decisions and architectural rationale.

Reusable Cursor prompts are located in:

```text
docs/cursor-prompts/

```
